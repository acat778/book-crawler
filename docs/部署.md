# acat-book-crawler 部署

## Docker 部署

```bash
# 构建镜像
docker build -t book-crawler:latest .

# 启动容器
docker run -d --name book-crawler --network host --restart unless-stopped \
  -v /opt/book-crawler-data:/app/data \
  -e NODE_ENV=production -e PORT=8609 \
  -e DATABASE_URL=mysql://root:123456@127.0.0.1:3306/acat_read \
  -e MONGO_URI=mongodb://127.0.0.1:27017/acat_dev \
  book-crawler:latest
```

## GitHub Actions CI/CD

Runner: 自托管，触发分支: `main`

| 阶段 | 操作 |
|------|------|
| 安装依赖 | `docker run node:22 npm install --no-audit` |
| 编译前端 | `docker run node:22 sh -c 'cd frontend && npm install && npm run build'` |
| 构建镜像 | `docker build -t book-crawler:latest .` |
| 部署 | `docker stop/rm book-crawler && docker run -d ...` |

配置：`.github/workflows/ci.yml`
