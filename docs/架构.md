# acat-book-crawler 架构

> Node.js 书籍爬虫工具 — Puppeteer + Cheerio + Express + WebSocket

## 技术栈

| 层级 | 技术 |
|------|------|
| 运行时 | Node.js ≥ 18 (ES Module) |
| 后端框架 | Express 4.x |
| 前端 | React 19 + TypeScript + Vite 8 |
| HTML 解析 | Cheerio |
| 浏览器自动化 | Puppeteer |
| HTTP 客户端 | Axios + lossless-json |
| WebSocket | Node.js 原生 WebSocket |

## 项目结构

```

├── src/
```

## 爬取流程

1. 搜索书籍（DuckDuckGo/站内搜索）
2. Puppeteer 获取书籍详情页 → Cheerio 解析元数据
3. 通过 crawler API 查找已有书籍 → 不存在则创建
4. 获取目录页 → 解析章节列表
5. 逐章爬取内容 → 通过 API 创建章节+段落
6. WebSocket 推送爬取状态

## 多站点适配

通过 `SiteAdapter` 基类实现适配器模式，每个站点继承并实现：
- URL 模式（书籍/章节/目录 URL 解析）
- HTML 解析规则（元数据/目录/正文提取）
- 搜索策略（站内搜索或外部搜索引擎）
- Puppeteer 交互（Cloudflare 绕过、懒加载内容展开）
