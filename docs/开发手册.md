# acat-book-crawler 开发手册

## 环境要求

- Node.js ≥ 18.0.0
- 后端 API 服务已运行（acat-book-gateway + 各微服务）

## 安装与运行

```bash
# 安装依赖
npm install && cd frontend && npm install && cd ..

# 开发模式
npm run dev                    # 后端 :8609（热重载）
npm run dev:frontend           # 前端 :5175（热更新）

# 生产模式
cd frontend && npm run build && cd ..
npm start                      # :8609
```

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `PORT` | `8609` | 服务端口 |
| `DATABASE_URL` | — | Prisma MySQL 连接串 |
| `MONGO_URI` | — | MongoDB 连接串 |
| `MINIO_ENDPOINT` | `http://localhost:9003` | MinIO/S3 地址 |
| `MINIO_BUCKET` | `acat-book` | 小说文件 Bucket |

## 编码规范

- **模块系统**：ES Module（`import`/`export`）
- **命名**：函数/变量 camelCase，类 PascalCase，文件 kebab-case
- **日志**：统一用 `console.log/warn/error`，带模块前缀 `[Crawler]`、`[ApiClient]`、`[Search]`
- **注释**：公开方法用 JSDoc，内部方法按需注释
- **配置**：所有环境变量通过 `config.js` 读取，不硬编码
- **数据操作**：所有数据通过 REST API 操作后端数据库，不直接连接数据库

## 添加新站点适配器

1. 在 `src/sites/` 下创建 `site-{站点名}.js`
2. 继承 `SiteAdapter` 基类，实现所有必需方法
3. 在 `src/sites/registry.js` 中注册新适配器
